# ae-snow-generator-script
 generate customisable snow using solid layers as particles

<p>to run, call:
generateSnow(numParticles, flakeVariation, speedVariation, depth);
// recommended ranges
// numParticles: 100-1,000
// flakeVariation: 0-20
// speedVariation: 0-25
// depth: 0-10,000
</p>
